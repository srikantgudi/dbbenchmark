# A benchmark application 

This is a respository of applications in C++, Go and PHP connecting to a `PostgreSQL` sample database.

### C++ App

This is an app in C++.

to build:
```
cd cpppgapp
make
```

to run:
```
./main
```

__Refer to `README.md` for more info.__

### Go app

This is an app in `Go`.

to build the app:

```
cd gopgapp
go mod tidy
go build .
```

to run:

```
./gopgapp
```

### PHP app

```
cd phpapp
php phppgapp.php
```
